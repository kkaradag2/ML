# Makine Öğrenmesi – Ders Notları

Bu repository, **makine öğrenmesi** konularını **Türkçe ders notları** halinde toparlamak amacıyla hazırlanmıştır.  
Amaç; konuları **temel matematik altyapısı ile**, **adım adım**, **örneklerle** ve **kaynak üretme** hedefiyle sunmaktır.

Notlar akademik bir ders akışına uygun olacak şekilde ilerler ve mümkün olduğunca
- sezgisel açıklamalar,
- sayısal örnekler,
- grafikler
ile desteklenir.
